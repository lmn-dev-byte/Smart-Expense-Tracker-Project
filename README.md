# Smart Expense Tracker

A React-based Expense Tracker application that helps users manage income and expenses.

## Features

- Add Transactions
- Track Income
- Track Expenses
- Calculate Balance
- Delete Transactions
- Local Storage Support
- Responsive Design

## Technologies

- React.js
- JavaScript
- CSS3
- Local Storage
